# Book Recommendations

My favorite technical and non-technical books.

## Programming

### The Rust Programming Language
*by Steve Klabnik and Carol Nichols*

The definitive guide to Rust. Clear explanations and great examples.

### Clean Code
*by Robert C. Martin*

Essential reading for writing maintainable code. Changed how I think about software design.

### Design Patterns
*by Gang of Four*

Classic book on reusable object-oriented design patterns.

## Computer Science

### Introduction to Algorithms (CLRS)
*by Cormen, Leiserson, Rivest, and Stein*

Comprehensive algorithms textbook. Dense but thorough.

### The Pragmatic Programmer
*by Andrew Hunt and David Thomas*

Practical advice for software development careers.

## Personal Development

### Deep Work
*by Cal Newport*

Strategies for focused work in a distracted world.

### Atomic Habits
*by James Clear*

How small changes can lead to remarkable results.

## Currently Reading
- "Designing Data-Intensive Applications" by Martin Kleppmann
- "The Phoenix Project" by Gene Kim
