# Welcome to MD_Filer

MD_Filer is a powerful markdown file management tool written in Rust.

## Features

- **Fast Search**: Quickly find notes by title, content, or tags
- **Tag Management**: Organize notes with tags and folders
- **Auto-indexing**: Automatic file indexing for quick access
- **Shortcuts**: Create shortcuts to frequently used files or URLs

## Getting Started

1. Run the app with `cargo run`
2. Use `list` to see all your notes
3. Use `search <query>` to find specific notes
4. Use `tags` to see all available tags and folders

Enjoy managing your markdown files!
